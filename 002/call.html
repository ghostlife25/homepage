ghostlife.org is an experiment in decentralized communal publishing. Its aim is to be a prototype for distributing information together with others, forming a community independent of any centralized platform.
It functions by joining together self-contained webpages into a webring each month. These webpages are not meant to be changed after publication. Being composed of individual, static files hosted on different servers means the construct can be decentralized and is possibly nomadic. These factors, along with the technical simplicity, could make this an easy and resilient mode of digital publishing.

There is no term yet for a self-contained webpage, so we propose the word webleaf*.

So, ghostlife.org is a webring of webleaves.

How to apply:

  1. Create a webpage with any static site generator or by writing HTML.
  2. Convert the webpage to a single HTML file using the free and open-source browser extension SingleFile. Now all media and stylesheets are encoded as one .html file. It is now a webleaf.
  3. Add the code snippet found on our website and adjust the colors of the widget to your liking.
  4. Upload your webleaf (for example, on Neocities, GitHub Pages, or your own server).
  5. Send us an email with the link to your leaf, how you’d like to be referred to, and how you found this open call.

Rules:

  • The webleaf should be smaller than 5 MB.
  • No hate speech, discriminatory language, or misinformation.
  • No links to social media or for-profit platforms.
  • Please refrain from sharing or mentioning ghostlife.org on social media or other for-profit platforms.

*The word works in three ways:
1 A hypertext document is referred to as a webpage. This is a bookbinding metaphor— in bookbinding, a page is printed onto a leaf.
2 In computer science, a data structure is described as a tree; a single file is called a leaf node.
3 The leaflet was the first medium for mass communication. Also called a pamphlet or flyer, single sheets of paper with information have been ubiquitous since the invention of the printing press. As of today, there is no agreed-upon digital equivalent of the printed leaflet.
